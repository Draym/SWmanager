Help : Planet

Tips:

The map of the world is huge. It is composed by :
- 6 galaxy.
- Each galaxy has 499 systems
- Each system has 15 planet, they are empty space if no player colonize them

Filters:

You can apply some filters to parse the list.
- You can search by galaxy.
- You can search by system range

Detail:

You can click on a Planet to see the detail of the owner profile.
